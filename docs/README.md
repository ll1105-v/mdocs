# AI 应用开发入门

这是一个面向个人网站场景整理的 AI 应用开发入门文档站。

如果你刚开始接触大模型、提示词、RAG、Agent 或 AI 产品原型，这里可以帮你快速建立一套整体认识。

## 你会在这里看到什么

- AI 应用开发的基本概念
- 一条适合初学者的学习路径
- 提示词设计的入门方法
- RAG 和 Agent 的区别与适用场景
- 一个 AI 小项目从想法到上线的基本流程

- 想用 Docsify 搭一个个人知识网站的人
